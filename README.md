# A-TECH-SALES-ANALYSIS-REPORT
This project analyzes sales performance for ATECH using an interactive Power BI dashboard. The goal is to uncover key business insights across regions, categories, and customer purchasing trends to support data-driven decision-making.

KEY KPIs

Sales: 4.59M
Profit: 572.79K
Quantity Sold: 76K

KEY INSIGHTS

 Sales show fluctuating trends with peak periods indicating seasonal demand
 West region is the top-performing region
 Technology category generates the highest revenue
 Phones and Chairs are top-selling sub-categories
 Discounts may be impacting overall profitability

 RECOMMENDATIONS
 
Optimize discount strategies to improve profit margins
Focus on high-performing products and categories
Improve sales in underperforming regions
Leverage peak sales periods for targeted campaigns
 
 TOOLS USED

Power Query & Power BI (Data Modeling, DAX, Dashboard Design)
